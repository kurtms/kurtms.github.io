About me

Projects

Solarri

I'm currently working with the Solarri Solar Car Data Team as we assemble the car's technical interface using an Android tablet and an Arduino. This was my first larger project working with a team as well as my first experience programming for Android. The Android interface is to poll data from the car's sensors and display the current data to the driver, all while recording it to a a database.

Game Programming

Following my first year of college, I wanted to make a larger project to reinforce the concepts we had learned. Using the low-level graphics and event library SDL, I created clones of both popular games 2048 and Pokemon. This was my first experience with event loops and management as well as graphical output beyond the command line.
